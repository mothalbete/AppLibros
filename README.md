📚 AppLibros — Aplicación web de gestión de libros
Aplicación sencilla desarrollada con HTML, CSS y JavaScript para gestionar una lista de libros.
Permite añadir, mostrar y eliminar libros de forma dinámica mediante manipulación del DOM.

✨ Funcionalidades
Añadir libros con título, autor y año.

Mostrar la lista completa de libros en pantalla.

Eliminar libros individualmente.

Validación básica de formularios.

Interfaz ligera y fácil de usar.

📁 Estructura del proyecto
Código
AppLibros/
│── index.html       # Interfaz principal de la aplicación
│── style.css        # Estilos de la interfaz
│── script.js        # Lógica de la aplicación (DOM, eventos, gestión de libros)
🚀 Cómo ejecutar
Clona el repositorio:

Código
git clone https://github.com/mothalbete/AppLibros
Abre index.html en tu navegador.

Empieza a añadir libros desde el formulario.

🛠️ Tecnologías utilizadas
HTML5

CSS3

JavaScript (DOM y eventos)
